# MERN Retail Management Platform

A full-stack retail management solution built using React.js, Node.js, Express.js, and MongoDB.

## Features

- JWT Authentication
- OTP Verification
- Category Management
- Product Management
- Image Upload
- Search & Filtering
- Role-Based Access Control

## Tech Stack

Frontend:
- React.js
- Material UI
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
